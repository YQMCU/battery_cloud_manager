# 后台程序

## 概述

后台采用ThinkPHP制作，有数据展示、地图展示(功能未开放)、远程参数修改、远程升级（需要额外加接口并修改程序，目前功能是上电自检测合宙云平台上传的程序版本和设备当前运行程序版本是否更新，然后进行远程程序升级）。
