# 云平台说明手册

## 使用方式

1. 准备服务器，将后台程序使用ftp上传到服务器路径中，部署web服务器的访问路径是根路径的public中

2. 使用node sever.js指令执行服务器程序

3. 使用mysql中的,source b4.sql命令创建数据库

4. 访问http://ip/login 登录后台 (ip为服务器的公网IP地址)
